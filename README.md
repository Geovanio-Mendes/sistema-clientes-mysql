# Sistema de Clientes e Pedidos

Sistema web CRUD completo desenvolvido com Python, Flask e MySQL.

## Tecnologias utilizadas
- Python 3
- Flask
- MySQL
- HTML + CSS
- JavaScript

## Funcionalidades
- Cadastrar, listar, buscar, atualizar e deletar clientes
- Cadastrar, listar, buscar, atualizar e deletar pedidos
- Interface web responsiva
- Conexão segura com banco de dados

## Como executar
1. Clone o repositório
2. Instale as dependências: `pip install flask mysql-connector-python python-dotenv`
3. Crie o arquivo `.env` com suas credenciais do banco
4. Execute: `python app.py`
5. Acesse: `http://localhost:5000`
